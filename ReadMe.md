# Disclaimer

This repository contains an inventory management system built with Java.

# Requirements

- JavaSE 15

# Instructions

### 1. Clone repository

```bash
git clone https://github.com/fanismahmalat/flowershop_inventory_management.git
```

### 2. Compile each class

```bash
javac <ClassName>.java
```

### 3. Run Main class

```bash
java Main
```
